# dmz-lab
Construyendo y Asegurando una Red con una Zona Desmilitarizada.

El objetivo del laboratorio fue diseñar e implementar una arquitectura de red segmentada con una DMZ que permitiera publicar un servidor web hacia Internet de forma controlada y segura. Para ello, se configuró el direccionamiento IP de las distintas redes, se aplicó NAT estático para exponer el servicio web y se implementaron ACLs para controlar el tráfico entre la red interna, la DMZ y la red externa.

Este repositorio incluye el archivo del proyecto en Packet Tracer con la topología completa, la configuración del router y la documentación del informe completa, las capturas de los procesos y las conclusiones obtenidas.
